# Data Sources

Synthetic source-style data for member digital journey measurement analytics.

- `entities.csv`: 36 digital journey records.
- `daily_metrics.csv`: 5,040 digital journey-day metric rows.
- `source_events.csv`: 760 operational events, QA checks, experiments, and stakeholder asks.
- `recommended_actions.csv`: 220 candidate actions with effort and expected lift.
